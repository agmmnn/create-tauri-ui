```bash
npx shadcn-svelte add textarea
```

1. Copy and paste the component source files linked at the top of this page into your project.

</ManualInstall>

## Usage

```svelte
<script lang="ts">
  import { Textarea } from '$components/ui/textarea';
</script>
```

```svelte

```
